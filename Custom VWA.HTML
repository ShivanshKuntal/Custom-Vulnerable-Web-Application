About SKVWA

SKVWA (Shivansh Kuntal Vulnerable Web App) is a visual clone intended for safe demonstrations. It intentionally omits server-side code, databases, and real vulnerabilities. Use it as a base to wire your own controlled training backend.

  • Pure HTML/CSS/JS (one file).
  • No external CDNs or trackers.
  • Hash-based client routing with graceful fallback.

How to Extend

  1. Split this file into views (e.g., /views/login.html) if needed.
  2. Add a backend of your choice (e.g., Node/Python/Go). Avoid deploying real vulnerabilities publicly.
  3. Gate any risky lab endpoints behind an auth wall on a private network/VM.

Security Note: Do not host intentionally vulnerable services on an exposed network.
